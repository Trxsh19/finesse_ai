# 💰 FinWise AI

An AI-powered personal finance assistant that helps users:
- Manage budgets
- Analyze debt
- Get AI financial advice (Groq LLaMA)
- Track financial health

---

## 🚀 Features
- User login system (SQLite)
- AI financial advisor (Groq API)
- Expense + debt parsing
- Real-time financial analysis
- Simple web UI (Streamlit)
